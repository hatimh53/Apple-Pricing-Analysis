🍎 Apple Product Pricing Analysis Dashboard

📊 Power BI dashboard analyzing Apple product pricing, discounts, and ratings across Amazon and Flipkart (2020-2026), with data cleaning done in Excel.


🛠️ Tools Used


📈 Excel — data cleaning & preprocessing
📊 Power BI — data modeling, DAX measures, dashboard design



🧹 Workflow

1. Excel (Cleaning)


Structured raw data into an Excel Table
Filled missing Sale_Event values with "Regular Day"
Fixed date formats & verified column data types


2. Power BI (Analysis)
Built around one core question:


How do discounts vary by platform, category, and sale season — and does that relate to ratings?




📌 Key Insights


📈 Discounts have steadily risen since 2020, sharpest jump after 2022
🔥 Sale events average ~31-37% discount vs ~21% on regular days
⭐ No link between review count and rating — 500 reviews or 3500+, ratings stay flat (4.44-4.46)
⌚ Watch & iPad get discounted more than iPhone & Mac
